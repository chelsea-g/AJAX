basicContext

basicContext is a plugin that allows you to add a context menu to (potentially) any element.
It is written completely in javascript but has the option to use jquery as well. It is useful
because the menu will never open outside the screen-area and will become scrollable if the height
of the menu is larger than the height of the browser window. It is easy to use and comes with
default themes that are easy to customize.
Plugin Source
Currently, output appears in the console log,
but each menu option can be programmed individually.
